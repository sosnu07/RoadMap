* 그냥 동영상 파일과 기본적으로 제공되는 영어 자막파일을 다운받고, 카카오 플레이어를 이용하여 실시간 번역 기능을 사용. 
  * 여러가지 번역 엔진이 있지만, 개인적으로는 카카오 번역과 파파고 번역을 추천
